# MoniMoni

A simple Python-based bank statement analyzer that helps users organize and inspect transaction data from bank statements.

## Features

* Read transactions from a bank statement file
* Sort transactions by date
* Sort transactions by payee
* Sort transactions by amount
* Find the maximum transaction amount
* Command-line interface for quick analysis

## Project Structure

```text
moni/
├── main.py
├── format.py
├── sort.py
├── README.md
└── statement.txt
```

## Requirements

* Python 3.x

## How to Run

1. Save your bank statement as `statement.txt` in the project folder.
2. Open a terminal in the project directory.
3. Run:

```bash
python main.py
```

4. Follow the on-screen instructions.

## Example Menu

```text
1. Sort by date
2. Sort by payee
3. Sort by amount
4. Find maximum transaction amount
```
